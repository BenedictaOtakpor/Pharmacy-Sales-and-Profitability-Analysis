# Pharmacy-Sales-and-Profitability-Analysis

### Project Overview
This project analyzes pharmacy sales data to evaluate revenue, profit, and margin trends across products, regions, and time. 
The dashboard enables stakeholders to identify top-performing products, uncover profitability gaps, and support data-driven decisions.

### Tools Used
- Power BI (DAX, Power Query,Data modelling)
